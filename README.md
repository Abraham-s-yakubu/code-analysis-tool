### `calculatePrice`

Calculates the final price of an item by applying an optional discount and a tax rate to a base price. The function performs input validation to ensure all arguments are numbers and within valid ranges, then returns the final price rounded to two decimal places.

#### Parameters

| Parameter   | Type     | Description                                                                                             |
| :---------- | :------- | :------------------------------------------------------------------------------------------------------ |
| `basePrice` | `number` | The initial price of the item. Must be a non-negative number.                                           |
| `taxRate`   | `number` | The tax rate to apply, expressed as a decimal (e.g., `0.05` for 5%). Must be a non-negative number.     |
| `discount`  | `number` | _(Optional)_ The discount rate to apply, expressed as a decimal (e.g., `0.10` for 10%). Defaults to `0`. Must be a number between `0` and `1` (inclusive). |

#### Returns

`number` - The final calculated price after applying the discount and tax, rounded to two decimal places.

#### Example Usage

```javascript
// Example 1: Calculate price with a 10% discount and 8% tax
const price1 = calculatePrice(100, 0.08, 0.10);
console.log(`Final price with discount and tax: $${price1}`); // Expected: $97.20 (100 * (1-0.10) * (1+0.08) = 90 * 1.08 = 97.2)

// Example 2: Calculate price with no discount (using default) and 5% tax
const price2 = calculatePrice(50, 0.05);
console.log(`Final price with no discount and tax: $${price2}`); // Expected: $52.50 (50 * (1-0) * (1+0.05) = 50 * 1.05 = 52.5)

// Example 3: Calculate price with a 25% discount and 0% tax
const price3 = calculatePrice(200, 0, 0.25);
console.log(`Final price with discount and no tax: $${price3}`); // Expected: $150.00 (200 * (1-0.25) * (1+0) = 150 * 1 = 150)

// Example 4: Invalid input (throws an error)
try {
  calculatePrice(100, -0.05, 0.1);
} catch (error) {
  console.error(`Error: ${error.message}`); // Expected: Error: Price and tax must be non-negative, and discount must be between 0 and 1.
}
```

### `formatDate`

The `formatDate` function takes a JavaScript `Date` object and returns a string representing the date in `YYYY-MM-DD` format. It ensures that single-digit months and days are padded with a leading zero. The function also includes robust validation, throwing an error if the provided input is not a valid `Date` object.

#### Parameters

| Parameter | Type   | Description                               |
| :-------- | :----- | :---------------------------------------- |
| `date`    | `Date` | The `Date` object to be formatted.        |

#### Returns

Returns a `string` representing the formatted date in `YYYY-MM-DD` format.

#### Example Usage

```javascript
// Get today's date and format it
const today = new Date();
const formattedToday = formatDate(today);
console.log(formattedToday); // Example: "2023-10-27"

// Format a specific date
const specificDate = new Date('2023-01-05T10:00:00Z');
const formattedSpecificDate = formatDate(specificDate);
console.log(formattedSpecificDate); // Output: "2023-01-05"

// Example with a single-digit month and day
const earlyDate = new Date('2024-03-08');
console.log(formatDate(earlyDate)); // Output: "2024-03-08"

// Handling invalid input
try {
  formatDate('not a date object');
} catch (error) {
  console.error(error.message); // Output: "Invalid date provided."
}

try {
  formatDate(new Date('invalid date string')); // This creates an "Invalid Date" object
} catch (error) {
  console.error(error.message); // Output: "Invalid date provided."
}
```

<!-- DOCS:START:slugify -->
This `slugify` function transforms a given string into a URL-friendly slug. It achieves this by converting the string to lowercase, replacing spaces with hyphens, normalizing accented characters to their basic ASCII equivalents, replacing ampersands with `-and-`, removing most non-alphanumeric characters, and finally cleaning up any redundant or leading/trailing hyphens to ensure a clean, consistent slug.

| Parameter | Type   | Description                               |
| :-------- | :----- | :---------------------------------------- |
| `text`    | `string` | The input string to be converted into a slug. |

Returns a `string` representing the URL-friendly slug.

```javascript
function slugify(text) {
    const a = 'àáâäæãåāăąçćčđďèéêëēėęěğǵḧîïíīįìłḿñńǹňôöòóœøōõőṕŕřßśšşșťțûüùúūǘůűųẃẍÿýžźż·/_,:;'
    const b = 'aaaaaaaaaacccddeeeeeeeegghiiiiiilmnnnnoooooooooprrsssssttuuuuuuuuuwxyyzzz------'
    const p = new RegExp(a.split('').join('|'), 'g')

    return text.toString().toLowerCase()
        .replace(/\s+/g, '-') // Replace spaces with -
        .replace(p, c => b.charAt(a.indexOf(c))) // Replace special characters
        .replace(/&/g, '-and-') // Replace & with 'and'
        .replace(/[^\w\-]+/g, '') // Remove all non-word chars
        .replace(/\-\-+/g, '-') // Replace multiple - with single -
        .replace(/^-+/, '') // Trim - from start of text
        .replace(/-+$/, '') // Trim - from end of text
}

const originalString1 = "Hello World! This is a Test String with special chars: àéüöß & more.";
const slug1 = slugify(originalString1);
console.log(slug1);
// Expected output: "hello-world-this-is-a-test-string-with-special-chars-aeuoss-and-more"

const originalString2 = "Another example --- with multiple spaces and --- hyphens ---";
const slug2 = slugify(originalString2);
console.log(slug2);
// Expected output: "another-example-with-multiple-spaces-and-hyphens"

const originalString3 = "  My Awesome Title!  ";
const slug3 = slugify(originalString3);
console.log(slug3);
// Expected output: "my-awesome-title"
```

### `capitalize`

This function takes a string as input and returns a new string with its first character converted to uppercase. If the input is not a string or is an empty string, the function gracefully handles these cases by returning an empty string.

#### Parameters

| Parameter | Type     | Description                               |
| :-------- | :------- | :---------------------------------------- |
| `text`    | `string` | The string to be capitalized.             |

#### Returns

Returns a `string`. A new string with the first character capitalized, or an empty string (`''`) if the input is not a string or is an empty string.

#### Example Usage

```javascript
// Capitalize a regular string
const originalString = "hello world";
const capitalized = capitalize(originalString);
console.log(capitalized); // Output: "Hello world"

// Handle an empty string
const emptyString = "";
const resultEmpty = capitalize(emptyString);
console.log(resultEmpty); // Output: ""

// Handle non-string input
const nonStringInput = 123;
const resultNonString = capitalize(nonStringInput);
console.log(resultNonString); // Output: ""

// Capitalize a string that is already capitalized
const alreadyCapitalized = "World";
const resultAlreadyCapitalized = capitalize(alreadyCapitalized);
console.log(resultAlreadyCapitalized); // Output: "World"
```

### `truncate`

Truncates a given string to a specified maximum length. If the string's length exceeds `maxLength`, it is cut off at `maxLength` characters and an ellipsis (`...`) is appended. If the input is not a string, an empty string is returned.

#### Parameters

| Parameter   | Type     | Description                                                                 |
| :---------- | :------- | :-------------------------------------------------------------------------- |
| `text`      | `string` | The string to be truncated.                                                 |
| `maxLength` | `number` | The maximum desired length of the string, including the ellipsis if appended. |

#### Returns

`string` - The truncated string with an ellipsis if necessary, or the original string if it's already within the `maxLength`, or an empty string if `text` is not a string.

#### Example Usage

```javascript
import { truncate } from './utils'; // Assuming truncate is in a utils file

console.log(truncate("Hello, world!", 5));
// Expected output: "Hello..."

console.log(truncate("Short", 10));
// Expected output: "Short"

console.log(truncate("A very long string that needs to be cut short.", 20));
// Expected output: "A very long string..."

console.log(truncate(12345, 5));
// Expected output: ""
```

### `isObjectEmpty`

This function determines if a given value is considered "empty". It returns `true` if the value is `null`, `undefined`, or a plain JavaScript object that has no own enumerable properties. It specifically distinguishes plain objects (those whose `constructor` is `Object`) from other object types like arrays, Dates, or objects created with `Object.create(null)`.

#### Parameters

| Parameter | Type | Description |
| :-------- | :--- | :---------- |
| `obj`     | `any` | The value to be evaluated for emptiness. |

#### Returns

`boolean`: `true` if `obj` is `null`, `undefined`, or an empty plain JavaScript object (i.e., `Object.keys(obj).length === 0` and `obj.constructor === Object`); otherwise, `false`.

#### Example Usage

```javascript
// Returns true for null or undefined
console.log(isObjectEmpty(null));      // true
console.log(isObjectEmpty(undefined)); // true

// Returns true for an empty plain object
console.log(isObjectEmpty({}));        // true

// Returns false for a non-empty plain object
console.log(isObjectEmpty({ a: 1 }));  // false

// Returns false for arrays (even empty ones) because their constructor is Array, not Object
console.log(isObjectEmpty([]));        // false
console.log(isObjectEmpty([1, 2]));    // false

// Returns false for other object types (e.g., Date)
console.log(isObjectEmpty(new Date()));// false

// Returns false for objects created with Object.create(null)
// because their constructor is undefined, not Object
console.log(isObjectEmpty(Object.create(null))); // false
```

### `getQueryParams`

This function parses a given URL string and extracts its query parameters into a plain JavaScript object. Each query parameter name becomes a key, and its corresponding value becomes the object's value. If the provided URL is invalid or has no query parameters, it returns an empty object.

#### Parameters

| Parameter | Type     | Description                                     |
| :-------- | :------- | :---------------------------------------------- |
| `url`     | `string` | The URL string from which to extract parameters. |

#### Returns

Returns a `Record<string, string>` (a plain JavaScript object) where keys are query parameter names and values are their corresponding string values. If the input URL is invalid or has no query parameters, an empty object (`{}`) is returned.

#### Example Usage

```javascript
// Example 1: URL with multiple query parameters
const url1 = "https://example.com/search?query=javascript&page=2&sort=desc";
const params1 = getQueryParams(url1);
console.log(params1);
// Expected output: { query: 'javascript', page: '2', sort: 'desc' }

// Example 2: URL with no query parameters
const url2 = "https://example.com/about";
const params2 = getQueryParams(url2);
console.log(params2);
// Expected output: {}

// Example 3: URL with encoded characters
const url3 = "https://example.com/items?category=Electronics&item=Laptop%20Pro";
const params3 = getQueryParams(url3);
console.log(params3);
// Expected output: { category: 'Electronics', item: 'Laptop Pro' }

// Example 4: Invalid URL
const url4 = "not-a-valid-url";
const params4 = getQueryParams(url4);
console.log(params4);
// Expected output: {} (and an error message logged to the console)
```

The `debounce` function creates and returns a new function that, when invoked, will execute the original function (`func`) only after a specified `wait` time has passed without any further invocations of the debounced function. This is particularly useful for limiting the rate at which a function is called, especially for events that fire rapidly (e.g., window resizing, input typing, button clicks), ensuring the function is only executed once the user has "finished" their action.

| Parameter | Type     | Description                                                                                             |
| :-------- | :------- | :------------------------------------------------------------------------------------------------------ |
| `func`    | `Function` | The function to be debounced. This function will be executed after the `wait` period.                   |
| `wait`    | `Number`   | The number of milliseconds to wait before executing `func`. The function will only execute after this delay has passed without further calls. |

**Returns**

`Function`: A new debounced function. When this returned function is called, it will delay the execution of `func` until `wait` milliseconds have passed without any subsequent calls. Any arguments passed to the debounced function will be forwarded to the original `func`.

**Example Usage**

```javascript
// A simple function to demonstrate debouncing
const logActivity = (action, value) => {
  console.log(`Action: ${action}, Value: ${value}, Timestamp: ${new Date().toLocaleTimeString()}`);
};

let inputCallCount = 0;

// Create a debounced version of logActivity for an input event
const debouncedInputHandler = debounce((event) => {
  inputCallCount++;
  logActivity('Input Changed', event.target.value);
  console.log(`Debounced input handler called ${inputCallCount} time(s).`);
}, 500); // Wait 500 milliseconds

// --- Simulate an input field ---
// In a real browser environment, you would attach this to an actual input element:
// const myInput = document.getElementById('mySearchInput');
// myInput.addEventListener('input', debouncedInputHandler);

// For this example, we'll simulate input events
console.log("Simulating rapid input typing...");

// Simulate typing 'h', 'e', 'l', 'l', 'o' quickly
debouncedInputHandler({ target: { value: 'h' } });
debouncedInputHandler({ target: { value: 'he' } });
debouncedInputHandler({ target: { value: 'hel' } });
debouncedInputHandler({ target: { value: 'hell' } });
debouncedInputHandler({ target: { value: 'hello' } }); // Only this one will trigger after 500ms pause

// Simulate a pause, then another character
setTimeout(() => {
  console.log("\nSimulating a pause, then another character...");
  debouncedInputHandler({ target: { value: 'hello w' } }); // This will trigger after 500ms from its call
}, 800);

// Simulate another rapid sequence after a longer pause
setTimeout(() => {
  console.log("\nSimulating another rapid sequence...");
  debouncedInputHandler({ target: { value: 'test' } });
  debouncedInputHandler({ target: { value: 'testing' } }); // Only this one will trigger after 500ms pause
}, 2000);
```

### `shuffle(array)`

This function shuffles the elements of an array randomly in place using the Fisher-Yates (Knuth) algorithm. It ensures that each permutation of the array elements is equally likely. If the provided input is not an array, it throws an error.

### Parameters

| Parameter | Type    | Description                                     |
| :-------- | :------ | :---------------------------------------------- |
| `array`   | `Array` | The array whose elements are to be shuffled.    |

### Returns

`Array` - The same input array instance, but with its elements randomly shuffled.

### Example Usage

```javascript
const numbers = [1, 2, 3, 4, 5];
console.log("Original array:", numbers);
// Expected output: Original array: [1, 2, 3, 4, 5]

const shuffledNumbers = shuffle(numbers);
console.log("Shuffled array:", shuffledNumbers);
// Expected output: Shuffled array: [?, ?, ?, ?, ?] (elements in a random order, e.g., [3, 1, 5, 2, 4])

console.log("Is it the same array instance?", numbers === shuffledNumbers);
// Expected output: Is it the same array instance? true

const fruits = ['apple', 'banana', 'cherry', 'date'];
shuffle(fruits);
console.log("Shuffled fruits:", fruits);
// Expected output: Shuffled fruits: ['?', '?', '?', '?'] (elements in a random order)

try {
  shuffle("this is not an array");
} catch (error) {
  console.error(error.message);
  // Expected output: The provided input must be an array.
}
```

### `generateUUID()`

This function generates a Universally Unique Identifier (UUID) string. It prioritizes the use of `crypto.randomUUID()` for secure and standard UUID generation in environments where it is available (e.g., modern browsers, Node.js). If `crypto.randomUUID()` is not present, it falls back to a pseudo-random generation method to produce a UUID-like string that adheres to the version 4 UUID format.

#### Parameters

| Parameter | Type | Description |
| :-------- | :--- | :---------- |
| None      |      | This function takes no parameters. |

#### Returns

-   **Type:** `string`
-   **Description:** A UUID string, typically in the format `xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx`.

#### Example Usage

```javascript
import { generateUUID } from './your-module-path'; // Adjust path as needed

const newUUID = generateUUID();
console.log(newUUID);
// Example output: "a1b2c3d4-e5f6-4789-abcd-ef0123456789"

const anotherUUID = generateUUID();
console.log(anotherUUID);
// Example output: "f0e9d8c7-b6a5-4432-1fed-cba987654321"
```
